# PAT Product Tabs for WooCommerce

Add lightweight, per-product custom tabs to WooCommerce product pages.

[Visit Price Action Tools](https://priceactiontools.com)

## What It Does

PAT Product Tabs for WooCommerce gives each product its own tab set. You can create only the tabs you need, control the label, reorder them, enable or disable each row, and add rich WYSIWYG content directly in the product editor.

This is a product-specific tab system, not a global or category-based tab plugin.

## Why Use It

- Per-product tab control
- No global tab rules
- WYSIWYG tab content
- Supports HTML, shortcodes, and embeds
- Simple reorder controls
- Built for WooCommerce
- Lightweight and easy to extend

## Installation

1. Upload the plugin to `wp-content/plugins/`.
2. Activate it in WordPress.
3. Make sure WooCommerce is active.
4. Edit a product and use the `Product Tabs` panel.

## How It Works

Each product stores its own tab data in post meta. On the frontend, the plugin injects only the tabs that are enabled and have content for that product.
